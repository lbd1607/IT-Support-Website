Remove activation prompts for Office 2016

Sometimes after installing and activating a Volume License copy of Office 2016, Windows will continue to prompt you to activate the product. This issue occurs because a version of Office has been installed on the system by the manufacturer or seller and was not completely removed before the Volume License version of Office 2016 was installed. This tutorial will teach you how to modify the registry to fix this error.

Caution: This task contains steps that will tell you how to modify the registry. Note that serious problems might occur if you modify the registry incorrectly. Please make sure that you follow these steps carefully. In addition, it is a good idea to back up the registry before you modify it so if a problem occurs, you can restore the registry from back-up. This tutorial will show you how to back-up a registry value in later steps.

For more information about how to back up and restore the registry, visit Microsoft’s website or contact your IT department.

Modify the registry

To resolve this problem, you will export the following registry keys to a save destination and then delete them from the computer. This process will require you to export and then delete two separate values from the registry.

  1. Close the Office activation (Let’s get started) window.
  2. Press Windows+R to bring up the Run dialog.
  3. In the Run dialog, type regedit and press Enter. A dialog box will pop-up asking if you want Registry Editor to make changes to your device. Click Yes.

  4. After the registry window pops-up, locate the following key in the registry by clicking on the arrow next to each of these folders: 
            
>HKEY_LOCAL_MACHINE 
>SOFTWARE 
>Wow6432Node 
>Microsoft 
>Office 
>16.0 
>Common 
>OEM
  5. Click the OEM value folder to select it. At the top left of the registry window, point to File, then click Export. Save the key in a place that you will be able to locate later if you need to restore it. Save this key as OEM1.

  6. After you have saved the registry key, ensure that the OEM folder is still selected. In the registry's top menu, point to Edit, then click Delete.
  7. A dialog box will pop-up to ask you if you want to delete this value. Click Yes.

  8. Locate the second key in the registry by clicking on the arrow next to each of these folders: 
    >HKEY_LOCAL_MACHINE 
>SOFTWARE 
>Microsoft 
>Office 
>16.0 
>Common 
>OEM
  9. Click the OEM value folder to select it. At the top left of the registry window, point to File, then click Export. Save the key in a place that you will be able to locate later if you need to restore it. Save this key as OEM2.

  10. After you have saved the registry key, ensure that the OEM folder is still selected. In the registry's top menu, point to Edit, then click Delete.
  11. A dialog box will pop-up to ask you if you want to delete this value. Click Yes.

You have successfully modified the registry. Windows should no longer prompt you to activate Office. You may now close all open windows and return to the desktop.
