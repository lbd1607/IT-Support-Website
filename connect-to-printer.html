Connect to a printer on the network and set a printer as default

Occasionally, you may need to connect your computer to a printer on the network so you can print remotely. This brief tutorial will teach you how to connect to a printer and how to set a printer as default.

Note: It is important that you are logged in to the computer that you will be printing from. Each user account (Windows profile) must be configured to connect to a specific printer on the network.

Connect to a printer

  1. Sign in to the computer with your BearPass user name and password.

    2. Note: If you are connecting to a printer from a device that isn't on the domain, such as your laptop, type @sgf after your user name in the User name field. For example BoomerBear@sgf.

  2. After you have signed in, press Windows+R to bring up the Run dialog.
  3. In the Run dialog, type \\coep and press Enter.

After the Printer Network window opens, double-click on the printer that you want to connect to. The Printer Installation dialog box opens. After Windows finishes installing printer drivers, the print queue window for the connected printer will open automatically.

You are now connected to a printer on the network. You may close all open windows and return to the desktop.

Set a printer as default

  1. Press Windows+S to bring up the Windows Search bar.
  2. In the Search bar, type "printer" and click Devices and Printers.

    3. After the Devices and Printers window pops-up, you should be able to see all of the printers that are connected to your computer. If your printer does not show up in this location, follow the Connect to a printer tutorial.

  3. To set a printer as your default, right-click on the printer. Click Set as default printer.
  4. A dialog box may pop-up to tell you that Windows will stop managing your default printer. Click OK.

In Printers, a green check-mark should appear over the printer icon to indicate that you have successfully set your printer as the default. You may close all open windows and return to the desktop.
